# Lacrei Saúde Challenge

Projeto desenvolvido como desafio técnico inspirado na plataforma Lacrei Saúde, com foco em acessibilidade, inclusão e experiência do usuário.

## 🚀 Deploy

A aplicação está disponível em:

https://lacrei-saude-challenge-psi.vercel.app

## 📂 Repositório

GitHub:

https://github.com/rafaelsnavarro/lacrei-saude-challenge

---

## 📋 Sobre o projeto

O objetivo deste projeto foi desenvolver uma aplicação web responsiva inspirada na proposta da Lacrei Saúde, oferecendo uma experiência acessível para pessoas que buscam profissionais de saúde.

A aplicação conta com:

* Página inicial institucional
* Página de profissionais
* Busca de profissionais por nome, especialidade ou cidade
* Tema claro e escuro
* Alto contraste
* Ajuste de tamanho de fonte
* Layout responsivo
* Boas práticas de acessibilidade

---

## 🛠 Tecnologias utilizadas

* Next.js
* React
* TypeScript
* Styled Components
* Context API
* Vercel

---

## ♿ Recursos de acessibilidade

Foram implementados recursos para melhorar a experiência de diferentes perfis de usuários:

* Alternância entre tema claro e escuro
* Modo de alto contraste
* Aumento de fonte (A+)
* Redução de fonte (A-)
* Skip Link para navegação por teclado
* Labels e atributos ARIA
* Estrutura semântica com HTML5

---

## 📱 Responsividade

A aplicação foi desenvolvida utilizando abordagem mobile-first e adaptada para:

* Smartphones
* Tablets
* Desktops

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/rafaelsnavarro/lacrei-saude-challenge.git
```

Entre na pasta:

```bash
cd lacrei-saude-challenge
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Acesse:

```txt
http://localhost:3000
```
